# SHIT APP PPT 叙事链

## 叙事结构

Hook -> Context -> Tension -> Product Idea -> Workflow -> Interaction -> Intelligence -> Retention -> Visual Logic -> Roadmap / Closing

## 逐段目标

1. **Hook**  
   用一句话和强视觉让观众记住：这是一个把坏情绪冲进宇宙下水道的产品。

2. **Context**  
   当代年轻人每天都在处理碎片化压力：bug、会议、DDL、社死、失眠、钱包空。

3. **Tension**  
   普通情绪工具太严肃，用户情绪崩溃时不想被说教，只想先有一个出口。

4. **Product Idea**  
   SHIT APP 把情绪变成可互动的彩色污渍，然后通过夸张冲洗仪式释放。

5. **Workflow**  
   选择模式、输入坏情绪、互动释放、冲水动画、小票反馈、AI 推荐玩法，形成完整闭环。

6. **Interaction**  
   点击、拖拽、摇晃、冲走，让用户感觉“这坨东西正在被处理”。

7. **Intelligence**  
   AI 不做诊断，而是推荐 3-5 分钟可执行的小玩法。

8. **Retention**  
   日历、统计、成就让用户看见自己处理过的情绪，并愿意回来。

9. **Visual Logic**  
   孟菲斯风格不是装饰，而是把负面情绪去严肃化、实体化、游戏化。

10. **Roadmap / Closing**  
    v1.0 先完成冲洗闭环，后续加入日历、成就、同步与个性化。

## 核心观点

SHIT APP 不是让用户“变积极”，而是给用户一个荒诞、彩色、可交互的坏情绪出口。
