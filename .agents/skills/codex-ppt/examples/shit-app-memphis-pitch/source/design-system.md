# SHIT APP PPT 视觉系统

## 视觉气质

整体为黑色舞台感模板，顶部保留白色品牌 logo 区，主体区域爆发高饱和孟菲斯图形。像一次“黑盒里的情绪冲洗秀”：克制的商业活动 header + 荒诞、彩色、玩具感的产品内容。

## 参考模板

使用 `assets/header-template.png` 作为 header 参考：

- 左上：白色 `AttraX | BREWTOWN 啤酒小镇`
- 右上：白色 `mixclub 万象会` 与 M 形标志
- 背景：纯黑
- 主体内容尽量不侵入顶部 logo 安全区

## 画面比例

- 16:9
- 推荐 1920×1080 或同等比例
- 安全边距：左右 110px；顶部 logo 区 110px 内不放主文案；底部 70px 不放小字

## 色彩

| 颜色 | 色值 | 角色 |
| --- | --- | --- |
| 深黑 | `#000000` | 主背景 |
| 白色 | `#FFFFFF` | logo、主标题、高对比文字 |
| 亮黄 | `#FFE600` | 主按钮、重点强调 |
| 热粉 | `#FF006E` | 情绪标签、危险信号 |
| 湖蓝 | `#00CFFF` | 冲水、冷静、完成 |
| 薄荷绿 | `#00FFB2` | 治愈、成功、重启 |
| 橙红 | `#FF6B35` | 爆炸、愤怒、高能量 |

## 字体方向

- 标题：超粗黑体、海报感、白色或亮黄，短句。
- 英文产品名：粗体无衬线，`SHIT APP` 必须清晰。
- 辅助标签：贴纸式短标签，尽量 2-5 字。
- 小票文字：等宽感，但不要过多小字。

## 统一页面结构

- 顶部为模板 logo 区，保持白色、简洁。
- 主体内容位于中下区域或画面中心。
- 每页一个视觉主物件：马桶、漩涡、情绪污渍、7 模式贴纸、流程装置、小票、日历、路线图。
- 页脚可出现极小页码，但不是必须。

## 孟菲斯图形语言

- 大号圆点、波点背景
- 锯齿闪电
- 粗黑描边几何贴纸
- 不规则色块
- 斜向条纹
- 图形爆炸碎片
- 夸张按钮和旋涡

## 图像风格

- 2.5D 玩具感 + 平面孟菲斯拼贴。
- 颜色像夜店霓虹海报，但主体信息清楚。
- 情绪对象可以是彩色污渍、小怪球、乱码泡泡、便便造型、黑云。
- 不要真实脏污或恶心写实，不要恐怖。

## 版式变体

1. **封面大视觉**：大标题 + 巨大情绪冲洗装置。
2. **三卡观点页**：3 个大卡片，每个卡片只有短标题。
3. **流程页**：横向或环形流程，5-6 个步骤。
4. **系统页**：中心装置 + 周围模块标签。
5. **数据页**：日历/图表被做成孟菲斯贴纸。
6. **路线图页**：三阶段或四阶段彩色轨道。

## 负面约束

- 不要白底商务 PPT。
- 不要柔和疗愈冥想风。
- 不要大段文字。
- 不要复杂真实图表。
- 不要把中文小字做得太密。
- 不要让装饰遮挡主标题。
- 不要出现随机乱码、拼错的产品名或伪 logo。
