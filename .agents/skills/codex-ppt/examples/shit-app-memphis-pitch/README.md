# SHIT APP Memphis Pitch Example

This example is a 10-page product pitch deck generated with Codex PPT.

The deck uses a black Memphis-style visual system and treats every slide as one complete AI-rendered page image. The workflow artifacts are included so you can inspect how the deck was planned before generation.

## Files

- `final/shit-app-memphis-pitch.pdf` — final PDF deck
- `final/shit-app-memphis-pitch.pptx` — PPTX wrapper containing the final slide images
- `slides/` — selected slide PNGs for preview
- `source/brief.md` — deck brief
- `source/storyline.md` — narrative chain
- `source/design-system.md` — visual system
- `source/storyboard.md` — slide-by-slide plan
- `source/qa.md` — QA log
