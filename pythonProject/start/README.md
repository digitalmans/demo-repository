# 用户认证系统

提供登录和注册功能的Web应用，登录成功后跳转到语音助手系统。

## 功能特性

- 🔐 **用户注册**: 创建新账号
- 🔑 **用户登录**: 安全登录验证
- 🎤 **语音助手**: 登录后访问语音识别和生成功能
- 🎨 **现代化UI**: 响应式设计，支持移动端
- 🔒 **Session管理**: 基于Flask Session的用户认证

## 安装步骤

### 1. 安装MySQL数据库

确保MySQL 9.5已安装并运行。

### 2. 配置数据库连接

编辑 `config.py` 文件，修改数据库配置：
```python
MYSQL_CONFIG = {
    'host': 'localhost',      # MySQL主机地址
    'port': 3306,             # MySQL端口
    'user': 'root',           # MySQL用户名（请根据实际情况修改）
    'password': 'xh050316',   # MySQL密码
    'database': 'voice_assistant_db',  # 数据库名称
    'charset': 'utf8mb4'
}
```

### 3. 安装Python依赖

```bash
pip install -r requirements.txt
```

### 4. 运行应用

```bash
python app.py
```

应用启动时会自动：
- 创建数据库（如果不存在）
- 创建用户表（如果不存在）

### 5. 访问应用

打开浏览器访问: `http://127.0.0.1:5001`

## 使用说明

### 注册新账号

1. 访问注册页面
2. 填写用户名（至少3个字符）
3. 填写密码（至少6个字符）
4. 确认密码
5. 可选填写邮箱
6. 点击注册

### 普通用户登录

1. 访问登录页面
2. 输入用户名和密码
3. 点击登录
4. 登录成功后自动跳转到语音助手页面

### 管理员登录

1. 访问登录页面，点击"管理员登录"链接
2. 或直接访问 `/admin/login`
3. 输入管理员用户名和密码
4. 登录成功后跳转到管理员后台

**默认管理员账号**:
- 用户名: `admin`
- 密码: `admin123`

⚠️ **重要**: 首次使用后请立即修改默认管理员密码！

### 管理员功能

管理员后台提供以下功能：
- 📊 **用户统计**: 查看总用户数、管理员数、普通用户数
- 👥 **用户管理**: 查看所有用户列表
- 🔄 **角色管理**: 修改用户角色（普通用户/管理员）
- 🗑️ **用户删除**: 删除用户账号（不能删除自己）

### 语音助手

登录成功后，会自动跳转到语音助手页面，可以：
- 进行语音识别（ASR）
- 进行语音生成（TTS）

## 项目结构

```

    ├── css/
    │   └── style.css     # 样式文件
    └── js/
        ├── login.js      # 登录逻辑
```

## 数据库结构

### users 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键，自增 |
| username | VARCHAR(50) | 用户名，唯一 |
| password | VARCHAR(255) | 密码（SHA256哈希） |
| email | VARCHAR(100) | 邮箱（可选） |
| role | VARCHAR(20) | 用户角色：user(普通用户) 或 admin(管理员) |
| created_at | TIMESTAMP | 创建时间 |
| last_login | TIMESTAMP | 最后登录时间 |

## 注意事项

1. **数据库配置**: 首次使用前，请确保在 `config.py` 中正确配置MySQL连接信息
2. **MySQL服务**: 确保MySQL服务正在运行
3. **用户权限**: 确保MySQL用户有创建数据库和表的权限
4. **密码安全**: 密码使用SHA256哈希存储，不存储明文
5. **生产环境**: 请更改 `config.py` 中的 `SECRET_KEY`
6. **语音助手**: 确保主应用的语音助手运行在 `http://127.0.0.1:5000`
7. **端口配置**: 如需修改端口，请同时修改 `voice_assistant.html` 中的iframe地址

## 安全建议

- 生产环境使用更强的密码哈希算法（如bcrypt）
- 使用HTTPS协议
- 实现密码强度验证
- 添加登录失败次数限制
- 使用数据库存储用户数据
